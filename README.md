# BFIT
App de Salud y Bienestar 
# BFIT — Borrador del Proyecto

## Descripción del proyecto
BFIT es una aplicación móvil diseñada para acompañar a las personas en el control y mantenimiento de su peso saludable. Integra el registro de peso, altura, circunferencia de cintura, sexo y estructura ósea. Calcula automáticamente el Índice de Masa Corporal (IMC) y una referencia combinada IMC+cintura para evaluar mejor el estado corporal y el riesgo metabólico.

## Exposición del problema
El aumento gradual de peso suele pasar inadvertido. La falta de seguimiento y de referencias claras dificulta la prevención del sobrepeso y la obesidad. BFIT propone monitoreo simple, interpretación guiada y recordatorios periódicos para fomentar hábitos saludables y evitar “sorpresas”.

## Plataforma
Primera versión en Android (Android Studio + Kotlin) por alcance y accesibilidad. Futuras extensiones multiplataforma (Flutter/React Native) son posibles.

## Interfaz de usuario e interfaz de administrador
- **Usuario:** formularios simples para datos físicos, cálculo automático, gráfico de evolución, mensajes motivacionales y recordatorios configurables.  
- **Administrador:** mantenimiento de tablas de referencia (p. ej., rangos IMC), supervisión técnica y gestión de datos (acceso restringido).

## Funcionalidad
- Registro/edición de perfil.  
- Carga de peso, altura, circunferencia de cintura.  
- Cálculo de IMC e interpretación según referencias.  
- Valor combinado IMC+cintura como indicador ampliado.  
- Inclusión de sexo y estructura ósea.  
- Recordatorios periódicos para nuevas mediciones.  
- Historial y evolución (gráficos simples).

## Diseño (wireframes o esquemas)
1. **Inicio/Login**  
2. **Panel principal** (último registro, IMC, gráfico de progreso)  
3. **Carga de datos** (peso, altura, cintura, sexo, estructura ósea)  
4. **Resultados** (IMC, rango, referencia IMC+cintura, recomendaciones)  
5. **Configuración** (perfil y recordatorios)
